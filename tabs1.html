Tab Animation

Each tab is animated using the jquery method "animate". They are animated by changing the top and opacity values via the function "tabAnimation1()".

Panel Reveal

Each tab is linked to an article panel. The panels are hidden using the css property display set to a value of none. The function "linkTabAndArticle()" causes a the class "active" to be added to the article div with the corresponding id, causing it to become visible.

Click

The tabs are also clickable. Currently, the function getHref() gathers the value of href and logs it to the console.

Problems

This program is far from ideal. The biggest problem is with the function "linkTabAndArticle()". I'm sure you would agree, this is one big ugly function. It's about as un-DRY as you can get. Ideally there should be a separate function to get the id values; then a function to link those values to the corresponding article panel, and a separate function to cause the panel to be revealed.

Also...

There are lots of little problems. I'm not to happy with the animation of the tabs. Sometimes they bouncy up and down if you mouse over them in just the wrong way. Also, I'd like to make the panels fade in rather than just appear abruptly.
